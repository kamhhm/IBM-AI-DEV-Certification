# Emotion Detection Lab

A web-based emotion detection app built with Watson NLP API and Flask. Detects five emotions from text input and identifies the dominant emotion.

## Main Learnings
- Real-time emotion analysis using Watson NLP
- Flask backend with JSON API endpoints
- Error handling for blank inputs (status code 400)
- Unit testing with Python unittest
